                      # Screen Recording Feature Description

                      We will only get the live screen that the user chooses to be casted.Only "Screen Mirroring" feature of this app will use screen recording. The purpose of using screen recording is to cast the user's live screen to a TV display device that supports the DLNA protocol. this TV device must be in the same home internal network as the Mac computer. Just like a Mac or iOS device casts the screen through the Apple Airplay protocol to other playback equipment .Screen Recording data will only used within "Screen Mirroring" feature, and we never share it to any third parties, We don’t record the screen into files and we don’t send any content to a server. We only temporarily store data in the memory of the device before it is streamed to TV. Upon streaming, the content is removed from memory.
