    # 自定义规则

    # 介绍

    自定义规则 是一项简单但强大的功能。通过简单的规则配置，可以实现各种出乎意料的效果，请在阅读完本篇帮助文档后发挥您的才智，实现自己需要的过滤效果吧。当然，不要忘了分享到微博、朋友圈，让朋友们赞叹您的天赋。

    注意⚠️：只有在 启用自定义规则 开关按钮打开的情况下，您在应用程序 规则 页面编写的规则才会被执行。

    # 限制

    1. 您可以在不付费订阅高级版本的情况下使用自定义规则功能，但是仅能使用2条规则。
    2. 当您付费订阅开通高级版全部功能后，我们不建议您过多的配置规则。

    # 规则类型

    类型 优先级 作用 补充说明
    号码白名单 10 直接放行该号码发送的短信
    关键字白名单 20 直接放行包含关键字的短信
    号码黑名单 30 直接将该号码发送的短信标记为 spam（垃圾短信）
    关键字黑名单 40 直接将包含关键字的短信标记为 spam（垃圾短信）

    # 规则优先级

    自定义规则 是高级功能，在整个短信智能分类引擎中处于第二优先级序列，仅次于 信任五位数号码 的优先级。在 自定义规则 处理引擎中，不同的规则类型拥有不同的优先级序列。优先级数值越小的规则类型拥有越高的优先级。

    同类型的 自定义规则 在处理引擎中的处理优先级则取决于它在应用界面上展示的序号，序号越小的优先级越高。

    # 管理自定义规则

    要启用 自定义规则 ，需要您在应用程序首页底部的引擎配置区域打开 启用自定义规则 开关按钮。

    要禁用 自定义规则 ，需要您在应用程序首页底部的引擎配置区域关闭 启用自定义规则 开关按钮。

    # 增删自定义规则

    # 查看规则列表

    点击应用程序底部的导航栏 规则 可以切换到规则管理页面。该页面展示了当前所有 自定义规则 的清单。您可以在这个页面查看、创建、编辑、删除 自定义规则。

    # 创建规则

    点击 规则 页面标题 规则 右侧的 圆形加号 ⊕ 按钮，会弹出如下图所示的创建规则窗口。

    创建 自定义规则 时默认规则类型时 关键字黑名单，轻触规则类型文字，通过上下滑动可以改变将要创建的规则类型。设定规则类型后，您可以在窗口的文本输入框中填写您的规则内容。填写完成后，点击确定按钮会保存这条规则并立即生效。

    # 编辑规则

    按住规则所在的行，向左侧滑动，在拉出的菜单中点击蓝色 编辑 按钮，会弹出如下图所示的编辑窗口。 在编辑窗口中，您可以修改这条规则的规则类型和规则内容，编辑完成后，点击确定按钮会保存这条规则并立即生效。

    # 删除规则

    按住规则所在的行，向左侧滑动，在拉出的菜单中点击蓝色 删除 按钮，会立即删除这条规则，并立即生效。
